# 📊 Data Analytics Project

## Overview

This project demonstrates a complete Data Analytics workflow, transforming raw data into actionable business insights. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard creation in Power BI, and presentation of findings using Gamma.

The goal is to identify trends, patterns, and key performance indicators (KPIs) that support data-driven decision-making.

---

## Dataset

- Source: Customer Behaviour Dataset
- Format: CSV / Database
- Domain: Customer Analytics
- Purpose: Analyze customer behavior, trends, and business performance metrics.

## Project Workflow

### 1. Data Loading
- Imported dataset into Python.
- Verified data quality and structure.

### 2. Exploratory Data Analysis (EDA)
- Generated descriptive statistics.
- Analyzed distributions and relationships.
- Identified trends and anomalies.

### 3. Data Cleaning
- Handled missing values.
- Removed duplicates.
- Corrected data types.
- Standardized data formats.

### 4. SQL Analysis
- Loaded cleaned data into the database.
- Performed filtering, aggregation, joins, and KPI calculations.
- Extracted business insights using SQL queries.

### 5. Power BI Dashboard
- Connected Power BI to the database.
- Built interactive dashboards.
- Created KPIs, charts, and slicers.
- Enabled dynamic data exploration.

### 6. Reporting & Presentation
- Documented findings and recommendations.
- Created a professional presentation using Gamma.
